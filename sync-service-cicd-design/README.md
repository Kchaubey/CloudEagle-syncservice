# Sync Service - CI/CD Design

This repository contains the CI/CD pipeline design and deployment strategy for the Spring Boot based `sync-service`.

## Tech Stack

- Spring Boot
- MongoDB
- Jenkins
- Docker
- GCP VM
- Nginx
- GitHub

## Environments

- QA
- Staging
- Production

## Features

- Multi-environment deployment
- Jenkins CI/CD pipeline
- Zero downtime deployment
- Rollback mechanism
- Environment specific configurations
- Secret management strategy
